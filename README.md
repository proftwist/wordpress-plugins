# Wordpress-плагины

В этом репо будут wordpress-плагины Владимира Бычко.

GitHub Commit Chart, отображает диграмму коммитов в виде гуттенберговского блока.

Megalinks, отображает красивые тултипы у ссылок на внутренние посты и страницы.

Typo Reporter, позволяет отправлять информацию об опечатках по Контрол Энтер.

Post Wall, отображает стенку постов сайта в виде календаря с активностью по дням.

## Локализация

Для корректной работы локализации в плагинах необходимо компилировать JSON-файлы из PO-файлов с помощью команды:
`wp i18n make-json languages --no-purge`

Это обеспечит правильное отображение локализованных строк в JavaScript-файлах плагинов.
