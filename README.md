# Wordpress-плагины

В этом репо будут wordpress-плагины Владимира Бычко.

Пока тут один плагин, GitHub Commit Chart, он отображает диграмму коммитов в виде гуттенберговского блока.

## Версия 1.4.0 - Добавлена поддержка шорткодов

В новой версии 1.4.0 добавлена поддержка шорткодов для отображения диаграммы коммитов в классическом редакторе WordPress.

### Синтаксис шорткода:
```
[github-c github_profile="username"]
```

Где `username` - это имя пользователя на GitHub. Если параметр `github_profile` не указан, будет использоваться значение из глобальных настроек плагина.

Пример использования:
```
[github-c github_profile="vladimirbychko"]
```

## Версия 1.5.0 - Добавлена возможность выбора тега заголовка

В новой версии 1.5.0 добавлена возможность выбора тега заголовка для блока диаграммы коммитов в редакторе Gutenberg. Теперь можно выбрать один из следующих вариантов:

- H2
- H3 (по умолчанию)
- H4
- Обычный текст (центрированный текст без тега заголовка)

Все варианты отображаются с центрированием для лучшей визуальной презентации.
