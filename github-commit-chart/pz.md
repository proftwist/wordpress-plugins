# GitHub Commit Chart

Плагин для WordPress, который отображает диаграмму коммитов пользователя GitHub в виде блока Gutenberg.

## Описание

Этот плагин позволяет отображать активность коммитов пользователя GitHub в виде цветной тепловой карты, аналогично тому, как это сделано на сайте GitHub. Диаграмма показывает интенсивность активности пользователя по дням за последний год.

## Установка

1. Скачайте плагин и загрузите его в директорию `/wp-content/plugins/` вашего сайта WordPress
2. Активируйте плагин через меню "Плагины" в панели администратора WordPress
3. Перейдите в раздел "Настройки" -> "Git-диаграмма" и ваш никнейм на GitHub

## Использование

После установки и настройки плагина вы можете добавить диаграмму коммитов на любую страницу или запись с помощью блока Gutenberg:

1. Создайте новую страницу или запись или отредактируйте существующую
2. В редакторе блоков найдите блок "Git-диаграмма"
3. Добавьте блок на страницу
4. При необходимости укажите путь к профилю GitHub в настройках блока
5. Опубликуйте или обновите страницу/запись

## Настройка

Плагин имеет страницу настроек, доступную через "Настройки" -> "Git-диаграмма" в панели администратора. На этой странице вы можете указать путь к профилю GitHub, который будет использоваться по умолчанию для всех блоков диаграмм.

## Как это работает

1. Плагин получает данные о коммитах пользователя через GitHub API
2. Данные кэшируются на один час для уменьшения количества запросов к API
3. Статистика коммитов отображается в виде тепловой карты, где каждый квадратик представляет один день
4. Цвет квадратика зависит от количества коммитов в этот день:
   - Серый (#ebedf0) - нет коммитов
   - Светло-зеленый (#c6e48b) - низкая активность
   - Зеленый (#7bc96f) - средняя активность
   - Темно-зеленый (#239a3b) - высокая активность
   - Очень темно-зеленый (#196127) - очень высокая активность

## Технические детали

- Плагин использует AJAX для загрузки данных о коммитах
- Все данные кэшируются с помощью WordPress Transients API
- Диаграмма отображается с помощью JavaScript и CSS
- Поддерживается отображение месяцев и дней недели
- Используется моноширинный шрифт для ровного отображения элементов диаграммы

## Совместимость

Плагин совместим с WordPress 5.0 и выше, так как использует блоки Gutenberg.
