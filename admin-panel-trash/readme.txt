Управление элементами админ-панели WordPress. Отключайте ненужные элементы для оптимизации рабочего пространства.

== Description ==

Плагин Admin Panel Trash позволяет легко управлять элементами админ-панели WordPress. Вы можете просматривать все доступные элементы и временно отключать те, которые вам не нужны.

**Основные возможности:**

* Просмотр всех элементов админ-панели
* Отключение ненужных элементов
* Включение ранее отключенных элементов
* Проверка доступа к файлу functions.php
* Поддержка мультиязычности

== Installation ==

1. Upload the `admin-panel-trash` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings → Admin Panel Trash to start managing admin bar items

== Frequently Asked Questions ==

= Как это работает? =

Плагин добавляет код в файл functions.php вашей темы для отключения выбранных элементов админ-панели.

= Безопасно ли это? =

Да, плагин только добавляет стандартные функции WordPress для управления админ-панелью.

= Что делать если файл functions.php недоступен для записи? =

Вам нужно изменить права доступа к файлу или вручную добавить код из уведомлений плагина.
