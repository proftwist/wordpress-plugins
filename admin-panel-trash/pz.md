# Admin Panel Trash - Обновление до версии 2.0.0

## Описание плагина
Плагин Admin Panel Trash позволяет управлять элементами верхней панели WordPress. Пользователи могут временно отключать ненужные элементы админ-панели для упрощения интерфейса.

## Структура плагина
- admin-panel-trash.php - основной файл плагина
- languages/ - файлы переводов
- assets/ - CSS и JavaScript файлы
- includes/class-assets-manager.php - менеджер ресурсов

## Безопасность
- Все выводы экранируются функциями esc_html_e()
- Входные данные санитизируются
- Используются nonce для AJAX-запросов
- Применяется WP_Filesystem API

## Локализация
Плагин поддерживает русский и английский языки. Файлы переводов обновлены в соответствии с текущим кодом.
