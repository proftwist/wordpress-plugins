Гутенберговский блок для текста с боковым изображением.

== Description ==

Плагин добавляет гутенберговский блок "Text with Side", который позволяет создать блок текста с опциональным боковым изображением.

Особенности:
* Текст с поддержкой форматирования
* Опциональное боковое изображение
* Настройка позиции (слева/справа)
* Настройка ссылки на изображение
* Настройка ширины блока
* Полная локализация (русский/английский)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/text-with-side` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Text with Side' block in the Gutenberg editor
