# Text with Side - Gutenberg Block

## Описание доработки

**Дата обновления:** 27.11.2025
**Версия:** 1.1.0

### Основная доработка: Улучшение редактора блока

Плагин "Text with Side" был улучшен для обеспечения лучшего опыта редактирования в редакторе Gutenberg. Основные изменения касаются визуального отображения блока в редакторе.

### Что изменено:

#### В редакторе:
- **Блок теперь отображается в поле контента** с шириной 100% и максимальной шириной 400px (вместо позиционирования на полях)
- **Убраны float и отрицательные margin** - блок ведёт себя как обычный элемент контента
- **Увеличена максимальная ширина изображения** до 300px для удобства просмотра
- **Добавлена информативная подсказка** о том, как блок будет выглядеть на фронтенде
- **Добавлен ::before псевдоэлемент** с подписью позиции блока (Left Side Block / Right Side Block)
- **Улучшена визуализация** с границами, фоном и цветными акцентами

#### На фронтенде:
- **Поведение не изменилось** - блоки по-прежнему позиционируются на полях основного контента
- **Сохранилось float-позиционирование** с отрицательными margin для размещения в полях
- **Адаптивный дизайн** остался без изменений

### Функциональность блока:

1. **Текстовый редактор** - полноценная поддержка форматирования
2. **Управление изображениями** - загрузка и удаление через Media Upload
3. **Настройки позиции** - выбор стороны (слева/справа) для фронтенда
4. **Варианты ссылок** - отсутствие ссылок, ссылка на медиафайл, ссылка на страницу вложения
5. **Настройка ширины** - возможность задать ширину изображения на фронтенде
6. **Адаптивность** - корректная работа на всех устройствах

### Технические детали:

- **Упрощённая архитектура** - удалён сложный CSS для позиционирования в редакторе
- **Улучшенная стабильность** в работе с редактором Gutenberg
- **Локализация** - полная поддержка русского и английского языков
- **Совместимость** - работает с различными темами WordPress

### Структура файлов:

- `text-with-side.php` - основной файл плагина
- `src/index.js` - JavaScript код Gutenberg блока
- `assets/editor.css` - стили для редактора (обновлены)
- `assets/frontend.css` - стили для фронтенда
- `package.json` - конфигурация для сборки
- `release-history.json` - история версий (обновлена)

### Версия: 1.1.0

Эта версия фокусируется на улучшении пользовательского опыта в редакторе, делая редактирование блоков более интуитивным и предсказуемым, при этом сохраняя функциональность позиционирования на фронтенде.
