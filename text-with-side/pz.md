### Пояснительная записка

#### Описание
Блок "Text with Side" (text-with-side) - это Gutenberg-блок для WordPress, который позволяет отображать текст с боковым изображением на полях (margins) основного контента. Версия 1.1.0 представляет полностью переработанную архитектуру с упрощенным и стабильным позиционированием.

#### Основные возможности
- **Упрощенное позиционирование**: Использует проверенное CSS float-позиционирование с отрицательными margin для выхода на поля
- **Боковое изображение**: Поддержка загрузки изображений через Media Library с возможностью настройки ширины
- **Ссылки на изображение**: Возможность ссылки на исходный файл медиа или страницу вложения
- **Позиция**: Выбор левой или правой стороны
- **Адаптивность**: На мобильных устройствах блок переходит в обычный поток с горизонтальным расположением
- **Редактор**: Полная совместимость с Gutenberg редактором

#### Ключевые изменения в версии 1.1.0
- **Упрощенная архитектура**: Удален сложный JavaScript для позиционирования
- **Стабильное позиционирование**: Переход на надежное CSS float-позиционирование
- **Улучшенная совместимость**: Лучшая работа с различными темами WordPress
- **Производительность**: Убраны лишние зависимости и JavaScript
- **Мобильная адаптивность**: Улучшенная отзывчивость на всех устройствах

#### Структура файлов
- `text-with-side.php` - основной файл плагина с классом TextWithSidePlugin
- `src/index.js` - JavaScript код Gutenberg-блока (упрощенная версия)
- `assets/frontend.css` - стили для фронтенда с float-позиционированием
- `assets/editor.css` - стили для редактора
- `package.json` - конфигурация для сборки
- `build/` - скомпилированные файлы (index.js, index.asset.php)
- `release-history.json` - история версий

#### Технические детали
- Версия: 1.1.0
- Зависимости: React, WordPress Block Editor, Components
- Поддержка локализации: домен 'text-with-side'
- Совместимость: WordPress 5.0+, Gutenberg
- Позиционирование: CSS float с отрицательными margin

#### Использование
1. Установите плагин
2. В редакторе Gutenberg найдите блок "Text with Side"
3. Добавьте изображение и текст
4. Настройте позицию (лево/право), ширину изображения и тип ссылки
5. Блок автоматически позиционируется на полях основного контента

#### Технические особенности позиционирования
- **Левые блоки**: `margin-left: -220px` для выхода на левое поле
- **Правые блоки**: `margin-right: -220px` для выхода на правое поле
- **Очистка float**: Используются `::after` псевдоэлементы
- **Адаптивность**: На мобильных переход в обычный поток

#### Преимущества версии 1.1.0
- Надежное позиционирование без JavaScript
- Совместимость с любыми темами WordPress
- Улучшенная производительность
- Простота поддержки и расширения
- Стабильная работа в редакторе и на фронтенде
