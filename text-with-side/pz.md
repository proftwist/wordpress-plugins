<content"># Текстовый блок с боковым изображением (Text with Side)

## Описание плагина
Gutenberg блок для отображения текста с боковым изображением, которое "плавает" в полях страницы на фронтенде.

## Функциональность
- **Позиционирование**: Изображение может располагаться слева или справа от текста
- **Управление изображениями**: Загрузка, замена и удаление изображений через медиатеку WordPress
- **Настройка ссылок**: Возможность сделать изображение ссылкой на медиафайл или страницу вложения
- **Адаптивность**: Настраиваемая ширина изображения
- **Локализация**: Полная поддержка русского и английского языков

## Техническая информация

### Файлы локализации
```
languages/
├── text-with-side.pot        # Шаблон переводов
├── text-with-side-ru_RU.po   # Русский перевод (исходный)
├── text-with-side-ru_RU.mo   # Русский перевод (скомпилированный)
├── text-with-side-ru_RU.json # Русский JSON для JavaScript
├── text-with-side-en_US.po   # Английский перевод (исходный)
├── text-with-side-en_US.mo   # Английский перевод (скомпилированный)
└── text-with-side-en_US.json # Английский JSON для JavaScript
```

### Основные файлы
- `text-with-side.php` - основной PHP файл плагина
- `src/index.js` - JavaScript код Gutenberg блока
- `assets/editor.css` - стили для редактора
- `assets/frontend.css` - стили для фронтенда
- `build/index.js` - скомпилированный JavaScript

### Атрибуты блока
- `content` - текстовое содержимое
- `imageId` - ID изображения в медиатеке
- `imageUrl` - URL изображения
- `imageAlt` - альтернативный текст
- `position` - позиция (left/right)
- `imageLink` - тип ссылки (none/media/attachment)
- `width` - ширина изображения

## Инструкции по использованию
1. В редакторе Gutenberg найдите блок "Текст с краю" (Text with Side)
2. Настройте позицию изображения через боковую панель
3. Загрузите изображение или выберите из медиатеки
4. Настройте тип ссылки при необходимости
5. Отрегулируйте ширину изображения
6. Введите текстовое содержимое
