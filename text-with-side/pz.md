# Пояснительная записка к плагину "Text with Side"

## Описание
Плагин "Text with Side" представляет собой гутенберговский блок для WordPress, который позволяет создавать текстовые блоки с опциональным боковым изображением.

## Назначение
Плагин предназначен для создания информативных и визуально привлекательных текстовых блоков, где изображение может располагаться слева или справа от текста. Это особенно полезно для:
- Блогов и статей
- Корпоративных сайтов
- Портфолио
- Информационных страниц

## Основные возможности

### Функциональность блока
- **Текстовый редактор**: Полноценный редактор с поддержкой форматирования (жирный, курсив, ссылки, списки)
- **Изображение**: Опциональное боковое изображение с возможностью удаления
- **Позиционирование**: Выбор расположения изображения (слева/справа)
- **Ссылки на изображение**: Три варианта ссылок:
  - Без ссылки
  - Ссылка на медиафайл
  - Ссылка на страницу вложения
- **Настройка ширины**: Возможность изменения ширины блока изображения

### Настройки
1. **Позиция изображения**:
   - Слева от текста
   - Справа от текста

2. **Ссылка на изображение**:
   - Отсутствует
   - Ссылка на медиафайл
   - Ссылка на страницу вложения

3. **Ширина изображения**:
   - Настраиваемое значение (по умолчанию 150px)

## Технические характеристики

### Требования
- WordPress 5.8 или выше
- PHP 7.4 или выше
- Поддержка гутенберговских блоков

### Архитектура
- **Backend**: PHP с использованием WordPress API
- **Frontend**: JavaScript (React/JSX) для редактора
- **Стили**: SCSS с автоматической компиляцией
- **Локализация**: Полная поддержка русского и английского языков

### Файловая структура
```
text-with-side/
├── text-with-side.php          # Основной файл плагина
├── package.json                # Конфигурация npm
├── readme.txt                  # Описание для WordPress.org
├── src/
│   ├── index.js               # Исходный код блока
│   ├── editor.scss            # Стили редактора
│   └── style.scss             # Стили фронтенда
├── build/                     # Скомпилированные файлы
└── languages/                 # Файлы перевода
    ├── text-with-side.pot     # Шаблон перевода
    ├── text-with-side-ru_RU.po # Русский перевод
    └── text-with-side-en_US.po # Английский перевод
```

- Проверка nonce для всех операций
- Санитизация пользовательского ввода
- Экранирование вывода для предотвращения XSS
- Проверка прав пользователя

### Производительность
- Минификация и оптимизация CSS/JS
- Ленивая загрузка изображений
- Оптимизированные запросы к базе данных
- Кэширование скомпилированных ресурсов

## Локализация
Плагин полностью локализован на русский и английский языки. Все строки интерфейса, сообщения об ошибках и описания переведены.

### Поддерживаемые языки
- Русский (ru_RU) - по умолчанию
- Английский (en_US)

## Установка и использование

### Установка
1. Загрузить файлы плагина в директорию `/wp-content/plugins/text-with-side/`
2. Активировать плагин через панель управления WordPress
3. Блок "Text with Side" будет доступен в редакторе Гутенберга

### Использование
1. Добавить блок "Text with Side" в редакторе
2. Ввести текст в текстовое поле
3. Опционально добавить изображение
4. Настроить позицию и параметры в боковой панели
5. Опубликовать пост/страницу
